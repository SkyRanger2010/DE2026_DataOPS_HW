===============================================
Примеры вызовов API /api/v1/predict (curl)
ДЗ по теме 24: Полноценный ML-сервис
===============================================

Сервис должен быть запущен: docker compose up
API: http://localhost:8899

Признаки (10 параметров diabetes): age, sex, bmi, bp, s1, s2, s3, s4, s5, s6

-----------------------------------------------
Вызов 1: первый тестовый пациент
-----------------------------------------------
curl -X POST "http://localhost:8890/api/v1/predict" ^
  -H "Content-Type: application/json" ^
  -d "{\"age\": 0.04, \"sex\": -0.05, \"bmi\": 0.06, \"bp\": -0.04, \"s1\": -0.03, \"s2\": -0.01, \"s3\": 0.02, \"s4\": -0.01, \"s5\": 0.01, \"s6\": -0.04}"

Ответ: {"predict": -0.25}

-----------------------------------------------
Вызов 2: второй тестовый пациент
-----------------------------------------------
curl -X POST "http://localhost:8890/api/v1/predict" ^
  -H "Content-Type: application/json" ^
  -d "{\"age\": -0.09, \"sex\": 0.04, \"bmi\": 0.05, \"bp\": 0.02, \"s1\": 0.01, \"s2\": -0.02, \"s3\": 0.01, \"s4\": 0.03, \"s5\": -0.01, \"s6\": 0.02}"

Ожидаемый ответ (пример): {"predict": 98.12}

-----------------------------------------------
Вызов 3: третий тестовый пациент
-----------------------------------------------
curl -X POST "http://localhost:8890/api/v1/predict" ^
  -H "Content-Type: application/json" ^
  -d "{\"age\": 0.10, \"sex\": -0.04, \"bmi\": -0.07, \"bp\": -0.02, \"s1\": 0.00, \"s2\": 0.02, \"s3\": -0.01, \"s4\": -0.02, \"s5\": 0.04, \"s6\": -0.03}"

Ожидаемый ответ (пример): {"predict": 201.33}

-----------------------------------------------
Для Linux/Mac используйте \ вместо ^ в curl:
-----------------------------------------------
curl -X POST "http://localhost:8890/api/v1/predict" \
  -H "Content-Type: application/json" \
  -d '{"age": 0.04, "sex": -0.05, "bmi": 0.06, "bp": -0.04, "s1": -0.03, "s2": -0.01, "s3": 0.02, "s4": -0.01, "s5": 0.01, "s6": -0.04}'
